## ⚠️ Important Notes

### System Requirements

- **Recommended RAM:** 16GB to 20GB allocated to Minecraft
- **Java:** Java 17+ required
- **OptiFine:** ❌ Incompatible (shaders via Oculus/Rubidium)

### Server Requirements

- **Minimum:** 14GB RAM
- **Recommended:** 16GB+ RAM for optimal performance
- **Expected Performance:** Stable 20 TPS with multiple players

### Before Playing

1. **🌐 Allocate Sufficient RAM** - This is a heavy modpack requiring 16-20GB
2. **☕ Use Java 17+** - Required for 1.20.1 Forge
3. **🎮 No OptiFine** - Use included Oculus/Rubidium for shaders
4. **📖 Check Quests** - Follow the quest book for guided progression
